PROJECT | 01

TechStock

Student project | 2025

TechStock is a web application developed as part of the course DT191G - Web Development with .NET. The application is built with ASP.NET Core MVC and utilizes Entity Framework Core for database interaction. It is designed to manage products and stock levels in a fictional tech store, offering both administrative functionality and a public product catalog.

TechStock includes features such as CRUD management for products, categories, and brands, pagination, filtering, and sorting, as well as a PWA (Progressive Web App) for installation on local devices. The system also provides a REST API for retrieving product data and user management via ASP.NET Identity.

Administrators can manage products and content through a responsive interface built with Razor Views (MVC) and Bootstrap, while visitors can browse products, view details, and access product data through the API.

BookFinder

Student project | 2025

BookFinder is a web application developed as part of the course DT210G - Advanced Frontend Development. Built with React and TypeScript, the application allows users to search for books and manage reviews, integrating the Google Books API for fetching book data and a custom backend API for authentication, reviews, and likes. PROJECT | 05

VOFF&MJAU - AnimalStore

Student project | 2024

AnimalStore is a RESTful web service developed for the DT193G - Fullstack Development with Frameworks course. Built with Hapi.js and MongoDB Atlas, it enables CRUD functionality for products and users in a pet store inventory system, featuring JWT authentication, Hapi Cookie authorization, and Mongoose validation. The API, deployed on Render, serves as the backend for a full-stack application.

VOFF&MJAU is the client-side application built with Vue.js, providing a responsive SPA for managing products. It features JWT authentication, Vue Router navigation, and Bootstrap styling, communicating seamlessly with the AnimalStore API. Published on Netlify, it offers an intuitive interface for inventory management.
